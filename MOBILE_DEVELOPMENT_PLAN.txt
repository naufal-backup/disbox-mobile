# Disbox Mobile Re-creation (Web-to-Mobile)

This project has been recreated using the latest Disbox Web codebase, optimized for mobile devices.
It uses Capacitor to wrap the web application into a native Android/iOS shell.

## Optimization Improvements:
- Removed Kotlin in favor of a unified React/Vite codebase.
- Fixed UI issues: Sidebar now correctly stays above the Music Bar on mobile.
- Improved mobile feel: Disabled pull-to-refresh overscroll and text selection.
- Performance: Added settings to disable animations for low-end devices.

## How to Build & Run:

1. **Install Dependencies:**
   Run `npm install` (or your preferred package manager).

2. **Add Mobile Platforms:**
   ```bash
   npx cap add android
   npx cap add ios
   ```

3. **Build the Web Project:**
   ```bash
   npm run build
   ```

4. **Sync to Capacitor:**
   ```bash
   npx cap sync
   ```

5. **Open in Android Studio / Xcode:**
   ```bash
   npx cap open android
   # or
   npx cap open ios
   ```

## Development Workflow:
To see changes on your device while developing:
1. Ensure your phone and computer are on the same Wi-Fi.
2. Update `capacitor.config.json` server section:
   ```json
   "server": {
     "url": "http://YOUR_COMPUTER_IP:5173",
     "cleartext": true
   }
   ```
3. Run `npm run dev -- --host`.
4. Run `npx cap sync`.
5. Run the app from Android Studio.
